Política de Privacidade

Aplicativo: PRINTVIEW • Data de Vigência: 01 de Outubro de 2025

Introdução

Esta Política de Privacidade descreve como o aplicativo PRINTVIEW ("o Aplicativo"), desenvolvido por WAI CODE SISTEMAS ("nós", "nosso" ou "a Empresa"), lida com suas informações pessoais.

O Aplicativo PRINTVIEW não coleta, armazena ou compartilha quaisquer dados pessoais dos usuários.

1. Coleta e Uso de Informações

O PRINTVIEW não coleta nenhum tipo de informação, seja automaticamente, fornecida pelo usuário ou por serviços de terceiros. Seu uso é totalmente seguro e não envolve armazenamento de dados pessoais.

2. Permissões Críticas Solicitadas e Uso Local

O PRINTVIEW solicita acesso a permissões do sistema Android para operar suas funcionalidades essenciais. No entanto, **nenhum dado acessado por estas permissões é coletado, armazenado ou transmitido** a nossos servidores ou terceiros. O uso é estritamente local e temporário no seu dispositivo:

  • Autenticação Biométrica (Face/Impressão Digital): Utilizada exclusivamente para **autenticar o usuário localmente** e liberar o acesso ao aplicativo ou a funcionalidades específicas do app, conforme configurado no seu dispositivo. Os dados biométricos são gerenciados e processados apenas pelo sistema operacional do seu celular. **Não temos acesso aos seus dados biométricos.**
  • Microfone: Necessário para permitir o recurso de **pesquisa por voz**. O áudio capturado é processado localmente no dispositivo para converter a fala em texto e, em seguida, o texto é usado para realizar a pesquisa interna. **O áudio não é gravado, armazenado ou transmitido para nossos servidores.**
  • Acesso à Internet/Rede: Necessário para **buscar atualizações do aplicativo** e permitir a **comunicação com a impressora** ou com serviços essenciais de impressão do sistema operacional.
  • Acesso à Memória/Armazenamento (Leitura de Mídia): Necessário para que você possa selecionar e carregar arquivos (como imagens ou documentos) que deseja visualizar e enviar para impressão.
  • Acesso a Dispositivos de Impressão (Bluetooth/NFC): Necessário para permitir a comunicação com impressoras e a realização da função principal do Aplicativo.

3. Compartilhamento e Divulgação de Informações

Como não coletamos informações, não há dados a serem compartilhados com terceiros.

4. Retenção de Dados

O PRINTVIEW não retém dados, pois não realiza coleta de informações.

5. Seus Direitos de Privacidade (LGPD/GDPR)

Como nenhum dado é coletado, não há necessidade de exercer direitos de acesso, retificação, exclusão ou oposição em relação ao PRINTVIEW.

6. Segurança dos Dados

A segurança de seus dados é preservada pelo simples fato de que o PRINTVIEW não coleta nenhuma informação.

7. Links para Outros Sites

O PRINTVIEW não contém links para sites externos.

8. Privacidade de Crianças

O Aplicativo pode ser utilizado por todas as idades, uma vez que não coleta informações pessoais de nenhum usuário.

9. Alterações a Esta Política de Privacidade

Podemos atualizar esta política caso o funcionamento do aplicativo seja alterado no futuro. A data de vigência no topo será atualizada sempre que ocorrer alguma modificação.
